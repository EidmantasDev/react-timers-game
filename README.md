# React timers game

A small game using core React functionality

## Run Locally

Clone the project

```bash
  git clone git@github.com:EidmantasDev/react-timers-game.git
```

Go to the project directory

```bash
  cd react-timers-game
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```
